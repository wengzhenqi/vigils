    自定义风险类型

    适合内部 token 或团队约定的密钥前缀。第一版使用“前缀 + 最小长度”，避免普通用户误写复杂规则。

      权限 / 守门清单

      • storage —— 仅保存自定义网站、模式和 UI 设置等元数据,不保存页面原文
      • scripting —— 对用户授权的自定义网站动态注入守门脚本
      • activeTab —— 供 popup 读取当前页状态和打开选项页使用
      • host_permissions:ChatGPT / Claude / Gemini / Perplexity 及已列入 manifest 的国内 AI 站点
      • optional_host_permissions:用户主动添加的 HTTPS 域名
      • 实时提示在网页输入框内显示;Options 仅用于模式和权限说明
      • CSP script-src 'self',禁 unsafe-eval / 内联 script
      • 原文永不入 chrome.storage / console.log(ADR 0009 §I-9.1)
      企业连接属于高级能力，普通用户不需要安装 Native Host 或桌面应用。
